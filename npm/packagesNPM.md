### npm ls scripts 
[link](https://www.npmjs.com/package/npm-ls-scripts)
```bash
# package qui permet de lister l'ensembles des scripts
> npm install npm-ls-scripts -g
> ls-scripts
# exemple 
NPM - ls scripts
---
start - node server.js
dist - webpack -p
---

# ou bien la methode la plus simple
> cat package.json
```
